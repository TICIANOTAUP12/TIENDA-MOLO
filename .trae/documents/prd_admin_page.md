## 1. Product Overview
Sistema de administración para gestionar anuncios publicitarios en un parque. Permite a los administradores controlar el contenido visual, precios y exportar información relevante.
- Solución para administrar imágenes de anuncios, precios y generar reportes desde un panel centralizado.
- Dirigido a administradores del parque que necesitan actualizar contenido y precios frecuentemente.

## 2. Core Features

### 2.1 User Roles
| Role | Registration Method | Core Permissions |
|------|---------------------|------------------|
| Admin | Manual creation by super admin | Full access: upload images, modify prices, export data, manage ads |
| Viewer | Read-only access | View dashboard and reports only |

### 2.2 Feature Module
Nuestro sistema de administración consiste en las siguientes páginas principales:
1. **Login page**: autenticación de administradores, recuperación de contraseña.
2. **Dashboard page**: vista general de estadísticas, anuncios activos, ingresos.
3. **Ads Management page**: lista de anuncios, subir/editar imágenes, activar/desactivar.
4. **Pricing page**: configuración de precios por tipo de anuncio, historial de cambios.
5. **Export page**: generar reportes en Excel/PDF, selección de rangos de fecha.
